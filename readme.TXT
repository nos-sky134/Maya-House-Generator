Maya House Generator
by Emily Chen 

Instructions: 
Place the script in the Maya script editor
when the code is run, a menu with sliders for width, height and depth of the house will appear
specify the value of the width, height and depth as any number between 1 and 5 
check "Include Chimmey" to generate a house with a chimmey
check "Include Windows" to generate a house with windows 
Press the "Generate House" button to generate a house with those parameters
Press the "Generate Random House" button to create a house with a random width, height and depth
